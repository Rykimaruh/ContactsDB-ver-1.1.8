Contactsdb-ver-1.1.8

An application made to read, write, update, delete database files from Access, SQL or Oracle.

View on GitHub Download .zip Download .tar.gz

Application that can read, write, update, delete tables within Database files. Currently able to do so with Access DB. Soon with SQL and Oracle files. The purpose of this is to make it easier to manage database for common users rather than having painful tasks without messing up the system.

Currently in developing the create tables, check connections and make compatibility with other databases.
